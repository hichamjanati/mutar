MuTaR: Multi-task Regression in Python¶

This is a collection of sparse models for multi-task regression. Mutar models fit regularized regression models on a sequence of related linear models (X_1, y_1) … (X_k, y_k).

Estimators¶

Mutar contains implementations of the following models:

  • GroupLasso: l1/l2 regularized regression with identical feature supports across tasks. (Yuan and Lin, J. R Statistical Society 2006)

  • DirtyModel Generalization of the Group Lasso with a partial overlap of features using a composite l1/l2 and l1 regularization (Jalali et al., NeurIPS 2010).

Installation¶

On a miniconda environment:

git clone https://github.com/hichamjanati/mutar
cd groupmne
python setup.py develop

Otherwise, we recommend creating this minimal conda env

conda env create --file environment.yml
